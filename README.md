## 📌 Description :
Ce projet est une application web développée avec Spring Boot et Thymeleaf pour gérer une école de musique.
Elle permet de gérer les instruments, les professeurs et les cours programmés.

## 🎯 Objectifs :
* Gérer les instruments disponibles  

* Gérer les professeurs et leurs spécialités

* Planifier les cours

* Associer chaque cours à un professeur

* Filtrer et analyser les données

## 🛠️ Technologies utilisées :
* Java 17
* Spring Boot
* Spring Data JPA (Hibernate)
* Thymeleaf
* MySQL
* Maven
